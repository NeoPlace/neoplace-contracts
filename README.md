# NeoPlace Smart Contract

### Technologies

* Node.js 8

* Solc ^0.4.x

* Truffle ^4.x

## Installation

1. Install Truffle globally.
    ```javascript
    npm install -g truffle
    ```

	
2. Run the development console.
    ```javascript
    truffle develop
    ```

4. Compile and migrate the smart contracts.
    ```javascript
    truffle compile
    migrate
    ```
